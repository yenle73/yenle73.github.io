Object Removal from Photos Website
This application was developed as part of the CS406 - Image Processing and Application course to provide a powerful tool for removing unwanted objects from photos. Leveraging the advanced Mask-Aware Transformer (MAT) model, the system is based on the groundbreaking research presented in MAT: Mask-Aware Transformer for Large Hole Image Inpainting (CVPR 2022 Best Paper Finalist, Oral).
Credit Risk Modeling
Developed an end-to-end implementation of regulatory-compliant credit risk models using Python and real-world data. This project demonstrates expertise in data pre-processing (Weight of Evidence, fine/coarse classing), Probability of Default (PD) modeling with logistic regression, and model evaluation (ROC, AUC, Gini Coefficient, K-S Test). It aligns with Basel II/III standards and includes building a credit risk scorecard and assessing population stability, showcasing how data science is applied in the banking industry.

Multi-Approach Movie Recommender System
The project is conducted for my Python Programming Course - IE221. An overview of the project is to build a movie recommendation system based on 3 approaches: content-based, sentiment-based, and collaborative-recommendation. For the first two methods, the project utilizes natural language processing techniques to convert text into vectors and calculates the similarity between vectors using the cosine similarity measure. For the collaborative-recommendation approach, I employed the KNN model from the scikit-learn library and used the Euclidean distance measure to calculate similarity.
Grammatical Structure Analysis for Vietnamese
This project focuses on implementing a dependency parsing model to analyze the dependency relationships between words in a sentence. Dependency parsing is a natural language processing (NLP) technique that examines the grammatical structure of a sentence by identifying and establishing hierarchical relationships between words, such as subject-verb or modifier-noun.
CSearch - Computer Vision Paper Retrieval System
This project involves designing and implementing a text retrieval system to efficiently rank and extract relevant academic papers in the field of computer vision. Leveraging data analysis and natural language processing (NLP) techniques, the system uses the BIM (Binary Independence Model) and BM25 (Best Match 25) algorithms to prioritize results.
